Human Machine Interface (HMI)

The HMI is used to monitor and interact with the industrial process and PLC.

OpenPLC WebInterface

The PLC is realized with OpenPLC, controls the industrial process over Modbus TCP/IP.

Simulation Visualizer

The visualizer provides a dynamic view of the simulation independent of network/sensor manipulation.
Open

Historian Dashboard

The historian uses the TICK stack. Chronograf is the dashboard component, it has a pre-configured view of the process, but it can be used to explore all kinds of data points collected from the simulation.
Open
